## Context

You want to create a new instance of the [Redis](https://redis.io/) platform. 

## Steps

1. Choose the _Get New Services_ in the _Marketplace_ menu.
2. Add selected service Redis.
3. Type new instance service name, choose the box for deployment, and confirm it.
4. Wait for request processing and see the details of the new Redis instance. 
5. Use the admin panel link and generated credentials for [Redis Commander](https://github.com/joeferner/redis-commander) â€“ Redis web management tool.
